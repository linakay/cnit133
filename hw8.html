REST – PokéAPI (Documentation page)

The Pokemon API provides access to an official database of Pokemon, abilities, in-game items, events, and other elements that are useful for developers. The API itself only allows access to the HTTP GET method. You cannot add, delete, or otherwise alter the database without administrative privileges. However, PokéAPI is totally open and free for anyone interested in creating a Pokemon app or program to use without an API key.

There are a variety of resources available through the PokéAPI, which can be used to create a program. A typical request is written GET/api/v2/{endpoint}/. This will return a JSON file. If the endpoint is the name of a type of resource, the API returns the items in that resource's list. Specifying a name or id as the endpoint will return relevant information and attributes about the requested item.

You can add optional paramaters to the request following a question mark. ?limit=40 will display 40 results matching your query on each page. Appending an offset to this parameter separated by an ampersand (&) &offset=40 will advance to the next set of results.

Try the PokéAPI out for yourself at https://pokeapi.co/.
